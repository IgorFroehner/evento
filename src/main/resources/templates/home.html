Artigos

Registra os artigos publicados no evento. Cada artigo possui um id e um título. Os artigos são relacionados à edição em que foram publicados através do campo edicaoid, que é uma chave estrangeira para a relação edicoes. Além disto, opcionalmente pode ser definido um tipo para o artigo através do campo tipoid que é uma chave estrangeira para a relação tipos.

Edições

Registra as edições anuais do evento, com id, ano, cidade e estado que ocorrem. Além disto, o campo qtdparticipantes registra o total de pessoas inscritas para assistir ao evento.

Tipos

Registra os tipos dos artigos, contendo os campos id e nome. Os nomes dos tipos podem ser: “Graduação”, “Pós-Graduação”, “Pesquisa”, e “Aplicações/Experiências”.

Autores

Registra os dados dos autores de artigos. Cada autor possui um id, nome e genero que pode ser ‘f’ ou ‘m’.
